About this project

This web app provides a friendly interface for scanning local IPv4 networks and managing authorized devices remotely.

Scan devices

Discover active hosts on your subnet, including IP address, hostname, MAC address, and vendor/type.

Export results

Save scan output as a CSV file for auditing, inventory, or further analysis.

Remote control

Send shutdown commands to managed Windows and Linux/macOS devices safely from the web UI.

Why use this tool?

  • Fast subnet scanning with host detail discovery.
  • Built-in CSV export for reporting.
  • Clear status messages and safe shutdown controls.
  • Designed for administrators and authorized network operators.
